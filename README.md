# DetailPro CRM — Автодетейлинг орталығына арналған автоматтандырылған басқару жүйесі

**DetailPro CRM** — автодетейлинг орталықтарының (барлық келіп түскен тапсырыстарды, көліктерді, шеберлер командасын және чек шығару процесін) жұмысын жеңілдетуге және автоматтандыруға арналған заманауи веб-платформа. Жоба **Flask (Python)** бэкенд құрылымында және **RESTful API** архитектурасы негізінде жасалған.


Жобаның негізгі мүмкіндіктері (Критерийлер)

1. **Канбан-доска (Тапсырыстар мониторингі):** Келіп түскен көліктерді статус бойынша ("Күтуде" және "Жұмыс үстінде") нақты уақытта бақылау.
2. **Автоматты Базамен Байланыс (AJAX/Fetch):** Жаңа көліктерді немесе командаға жаңа шеберді қосқан кезде, бетті қатырмай, мәліметтерді артқы фонда SQLite базасына бірден (`201 Created`) енгізу.
3. **Аяқталған жұмыстар архиві:** Тапсырыс дайын болғанда, оның бэкендтегі статусы `completed` болып өзгереді және ол автоматты түрде "Архив" кестесіне сүзіліп шығады (Jinja2 фильтрациясы).
4. **Автоматты Чек Басып Шығару (Print Logic):** Көлік иесіне дайын жұмыс бойынша барлық ақпаратты (Госномер, Марка, Қызмет түрі, Бағасы, Жауапты шебер) қамтитын фискалды емес чекті моментально печатьқа жіберу.
5. **Мультилингвалды жүйе (KZ/RU Локализация):** Интерфейсті қазақ және орыс тілдеріне бір батырмамен ауыстыру мүмкіндігі (`localStorage` қолдауымен).



Технологиялық стек (Tech Stack)

* **Backend:** Python, Flask Framework, Flask-SQLAlchemy, Flask-Login (Авторизация).
* **Database:** SQLite (SQL негізіндегі реляциялық деректер базасы).
* **Frontend:** HTML5, CSS3 (Премиум қара түсті дизайн), JavaScript (Вnative AJAX Fetch API).
